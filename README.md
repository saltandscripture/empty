# Salt & Scripture

**Wear the Word** — Beautiful jewelry that connects you to God's Word with a single tap.

## About

Salt & Scripture is a faith-based platform featuring NFC-enabled jewelry that links to scripture, paired with a full KJV Bible reader with highlighting, notes, bookmarks, and more.

## Features

- Full KJV Bible reader with all 66 books
- 5-color verse highlighting
- Personal notes on any verse
- Verse bookmarks
- Daily & random verse tap experience
- Links to other translations (BibleGateway, Bible.com, Blue Letter Bible, Literal Word NASB)
- Beautiful, mobile-friendly design

## Development

```bash
npm install
npm run dev
```

## Build

